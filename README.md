# 🌍 منصة البث المباشر لكأس العالم

## نظرة عامة
منصة بث احترافية ومتكاملة لمتابعة مباريات كأس العالم بجودة عالية وواجهة مستخدم جذابة.

## ✨ الميزات الرئيسية

### 👥 نظام المشتركين
- تسجيل دخول آمن للمشتركين
- إدارة حسابات المستخدمين
- تجربة بث سلسة بدون تقطع

### 🎛️ لوحة تحكم الأدمن
- إدارة شاملة للمشتركين
- إضافة وتعديل روابط البث M3U
- عرض الإحصائيات والتقارير

### 📺 نظام البث
- 5 قنوات بث مختلفة
- خيارات جودة متعددة (عالية/منخفضة)
- بث مباشر بدون تأخير

### 🎨 التصميم
- واجهة مستخدم جذابة وحديثة
- متجاوب تماماً (Desktop, Tablet, Mobile)
- ألوان تعكس روح كأس العالم

### 🔒 الأمان
- تشفير كلمات المرور
- التحقق من الهوية المتقدم
- حماية البيانات الشاملة

## 🚀 البدء السريع

### المتطلبات
- Node.js 14.0 أو أحدث
- npm أو yarn
- قاعدة بيانات MySQL

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/FAISAL-ABDULLAH1/streaming-platform.git

# الانتقال إلى مجلد المشروع
cd streaming-platform

# تثبيت المكتبات
cd backend && npm install
cd ../frontend && npm install
```

### إعدادات البيئة

```bash
# في مجلد backend، أنشئ ملف .env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=streaming_platform
JWT_SECRET=your_jwt_secret
PORT=5000
```

### تشغيل المشروع

```bash
# تشغيل Backend
cd backend
npm start

# في Terminal جديد، تشغيل Frontend
cd frontend
npm start
```

## 📊 واجهات API الرئيسية

### إدارة المشتركين
- `GET /api/subscribers` - عرض جميع المشتركين
- `POST /api/subscribers` - إضافة مشترك جديد
- `PUT /api/subscribers/:id` - تحديث بيانات المشترك
- `DELETE /api/subscribers/:id` - حذف مشترك

### إدارة روابط البث
- `GET /api/streams` - عرض روابط البث
- `POST /api/streams` - إضافة رابط بث جديد
- `PUT /api/streams/:id` - تحديث رابط البث
- `DELETE /api/streams/:id` - حذف رابط البث

2. **قناة الرياضة 2** - جودة عالية وعادية
3. **قناة البث المباشر 1** - جودة عالية وعادية
4. **قناة البث المباشر 2** - جودة عالية وعادية
